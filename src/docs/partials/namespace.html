Tempus Dominus uses and exposes a Namespace class for consistancy and easy reference.

These values are provide via tempusDominus.Namespace

NAME

Returns tempus-dominus

version

Returns the current version of the picker

dataKey

Returns td. This is used as a suffix for events e.g. change.td

Events

A class that provides Event namespaces such as change.td

Css

A class that provides CSS namespaces for the widget

ErrorMessages

A class that provides Error Messages namespaces such as change.td
